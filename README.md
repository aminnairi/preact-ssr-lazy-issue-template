# preact-ssr-lazy-issue-template

## Requirements

- Git
- Node
- NPM

## Clone

```bash
git clone https://github.com/aminnairi/preact-ssr-lazy-issue-template
cd preact-ssr-lazy-issue-template
```

## Installation

```bash
npm install
```

## Build

```bash
npm run build
```

## Start

```bash
npm start
```
